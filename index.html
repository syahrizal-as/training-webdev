Web Developer Roles

Orang yang bertugas merancang, membuat, mengembangkan, dan memelihara situs web atau aplikasi web.

Front End Developer

Menggunakan HTML, CSS, dan JavaScript untuk mendesain dan membangun elemen visual interaktif yang berhadapan langsung dengan pengguna.

Back End Developer

Fokus pada logika server, basis data, dan API. Menangani autentikasi, otorisasi, dan pemrosesan data di balik layar.

PHP Node.js Python

Full Stack Developer

Menguasai front-end dan back-end. Mampu bekerja di semua lapisan aplikasi, dari UI/UX hingga arsitektur server dan database.

Laravel Setup

Instalasi & Persiapan

> composer create-project laravel/laravel:^13.0 app-name
> cd app-name
> php artisan serve
  • Buka terminal di Visual Studio Code
  • Pastikan versi PHP & Composer sesuai
  • Jalankan perintah instalasi di atas
  • Akses http://localhost:8000

Konfigurasi Database

  • Jalankan Apache & MySQL di XAMPP
  • Buat database baru di phpMyAdmin
  • Buka file .env di VS Code
  • Sesuaikan DB_DATABASE
  • Jalankan php artisan migrate
